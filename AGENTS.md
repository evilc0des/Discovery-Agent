# Client Requirement Intake Agent

## Agent skills

### Issue tracker

Issues are tracked as local markdown files under `.scratch/` in this repo. See `docs/agents/issue-tracker.md`.

### Triage labels

Default label vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout. `CONTEXT.md` at the repo root, `docs/adr/` for architecture decisions. See `docs/agents/domain.md`.
